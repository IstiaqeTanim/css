CSS Selectors

In CSS, selectors are patterns used to select the element(s) you want to style.

Use our CSS Selector Tester to demonstrate the different selectors.



1. .class Selector

Definition

The .class selector selects elements with a specific class attribute.

Example

My name is Istiaqe.
I live in Dhaka.

2. .class1.class2 Selector

Definition

Selects all elements with both name1 and name2 set within its class attribute

Example

My best friend is Tomal.

My best friend is Salauddin.


3. #id Selector

Definition

Selects the element with id="firstname"

Example

Your name is Mustak.


4. * Selector

Definition

Selects all elements


5. element Selector

Definition

Selects all elements

Example

Lorem ipsum, dolor set emet.


6. element.class Selector

Definition

Selects all elements with class

Example

My best friend is Masud.

7. element,element Selector

Definition

Selects all elements with another elements

Example

My best friend is kawser.

8. element1~element2 Selector

Definition

Selects every < ul> element that is preceded by a element

Example

The first paragraph.

  • Coffee
  • Tea
  • Milk

Another list

  • Coffee
  • Tea
  • Milk

9. [attribute] Selector

Definition

Selects all elements with a target attribute.
Links with a target attribute is styled with a yellow background

Example

W3schools.com
Google.com
Microsoft.com

10. [attribute=value] Selector

Definition

Selects all elements with target="_blank"

Example

W3schools.com
Google.com
Microsoft.com
